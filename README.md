# Grubby 🐚

Grubby is a lightweight, organized command-line utility for Termux.

## Installation
Run these commands in your Termux:

pkg update && pkg upgrade -y
pkg install git python cowsay -y
git clone https://github.com/johnsonsetiawan17-svg/Grubby
cd Grubby
python3 grubby.py

## How to use
Once the script is running, type any of the commands from the list below.

## Commands
Type .help inside the tool to see the vertical menu:
* .help : Show this list
* .exit : Close Grubby
* .clear : Clear the screen
* .hello : Get a greeting
* .date : Current date
* .time : Current time
* .calc : Simple math (e.g., .calc 2+2)
* .cowsay : Make a cow say something
* .joke : A random programmer joke
* .sys : Show system info
