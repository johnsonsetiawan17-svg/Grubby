# Grubby 🐚

Grubby is a lightweight, command-based utility tool designed specifically for Termux. It provides quick access to system information, handy shortcuts, and fun terminal commands.

## Features
- 30+ built-in commands
- QoL features for faster navigation
- Lightweight and dependency-free (Python 3)
- Easy to use: Just type and go!

## Installation
Open Termux and run the following commands:

```bash
# Update packages
pkg update && pkg upgrade -y

# Install git and python
pkg install git python -y

# Clone the repository
git clone https://github.com/johnsonsetiawan17-svg/Grubby

# Enter the folder
cd Grubby

# Run the tool
python3 grubby.py
