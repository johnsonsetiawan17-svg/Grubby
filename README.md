# Grubby 🐚

Grubby is a clean, organized Terminal QoL (Quality of Life) tool built for Termux.

## Features
- **Organized Menu**: Vertical list design for better readability on mobile screens.
- **Argument Support**: Use commands like `.cowsay hello` or `.calc 5+5`.
- **Lightweight**: Zero external dependencies.

## Installation
```bash
pkg update && pkg upgrade -y
pkg install git python cowsay -y
git clone [https://github.com/johnsonsetiawan17-svg/Grubby](https://github.com/johnsonsetiawan17-svg/Grubby)
cd Grubby
python3 grubby.py
